Description:
A travel planning app where users can plan their upcoming trips and manage important factors like budget and itinerary.

Link to project:
https://travel-wishlist-app-3.onrender.com

How It's Made:
Tech used: HTML, CSS, JavaScript, Node.js, Express, MongoDB, Mongoose, Passport.js, EJS, Bootstrap

Optimizations:
During development, I performed extensive code cleanup to improve readability, consistency, and maintainability. This included reorganizing route handlers, removing redundant logic, and improving file structure to make future updates more efficient and scalable.

Lessons Learned:
Working with MongoDB Atlas taught me about cloud databases and the advantages of managed database services over local installations.
